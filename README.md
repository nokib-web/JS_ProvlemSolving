# JavaScript Problem Solving

A collection of fundamental JavaScript problems and solutions, designed to help learn and practice core programming concepts.

## Overview

This repository contains solutions to classic JavaScript programming problems, perfect for beginners learning algorithm fundamentals and JavaScript syntax.

## Problems Included

### 1. Reverse a String
Convert a string in reverse order.
- **Input:** `"hello"`
- **Output:** `"olleh"`

### 2. Count Vowels in a String
Count the number of vowels (a, e, i, o, u) in a given string.
- **Input:** `"programming"`
- **Output:** `3` (o, a, i)

### 3. Check for Palindrome
Determine if a string reads the same forwards and backwards.
- **Input:** `"madam"`
- **Output:** `true`
- **Input:** `"hello"`
- **Output:** `false`

### 4. Find the Maximum Number in an Array
Find the largest number in an array.
- **Input:** `[5, 1, 9, 3]`
- **Output:** `9`

### 5. Remove Duplicates from an Array
Remove duplicate numbers from an array.
- **Input:** `[1, 2, 2, 3, 4, 4]`
- **Output:** `[1, 2, 3, 4]`

### 6. Sum of All Numbers in an Array
Calculate the sum of all numbers in an array.
- **Input:** `[1, 2, 3, 4]`
- **Output:** `10`

### 7. Find Even Numbers in an Array
Filter out even numbers from an array.
- **Input:** `[1, 2, 3, 4, 5, 6]`
- **Output:** `[2, 4, 6]`

### 8. Capitalize First Letter of Each Word
Capitalize the first letter of every word in a sentence.
- **Input:** `"hello world"`
- **Output:** `"Hello World"`

### 9. Find the Factorial of a Number
Calculate the factorial of a non-negative integer.
- **Input:** `5`
- **Output:** `120`

### 10. PingPong Challenge
Iterate from 1 to 20. Return "Ping" for multiples of 3, "Pong" for multiples of 5, "PingPong" for multiples of both, and the number otherwise.
- **Input:** `None`
- **Output:** `[1, 2, "Ping", 4, "Pong", "Ping", ... "PingPong"]`

## How to Use

Run the index.js file to see all solutions in action:

```bash
node index.js
```

For detailed explanations of how each algorithm works, check the description.js file which includes comprehensive comments for each problem.

## Getting Started

1. Clone or download this repository
2. Open the files in your preferred code editor
3. Run `node index.js` to execute all solutions
4. Study the code and comments to understand the algorithms
